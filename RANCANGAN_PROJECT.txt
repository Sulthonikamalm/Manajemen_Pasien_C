        RANCANGAN SISTEM MANAJEMEN RUMAH SAKIT JIWA SULTHIM
                    Desktop Application dengan GTK+

INFORMASI PROJECT
================================================================================
Nama Project    : Sistem Manajemen Rumah Sakit Jiwa SultHim
Bahasa          : C
GUI Library     : GTK+ 3
Database        : CSV Files
Platform        : Windows (Cross-platform ready)
Semester        : 6


FITUR UTAMA
================================================================================

1. MANAJEMEN PASIEN
   - Create: Tambah data pasien baru
   - Read: Lihat daftar & detail pasien
   - Update: Edit data pasien
   - Delete: Hapus data pasien (dengan konfirmasi)
   
   Data Pasien:
   - ID Pasien (auto-increment)
   - Nama Lengkap
   - NIK
   - Jenis Kelamin (Laki-laki/Perempuan)
   - Tanggal Lahir / Usia
   - Alamat Lengkap
   - Nomor Telepon
   - Riwayat Penyakit
   - Tanggal Registrasi
   - Status (Aktif/Tidak Aktif)

   Fitur Tambahan:
   - Search: Cari berdasarkan nama/NIK
   - Filter: Filter berdasarkan jenis kelamin, status, usia
   - Sorting: Urutkan berdasarkan nama, usia, tanggal registrasi
   - Pagination: Tampilkan data per halaman (20 data/halaman)


2. REKAM MEDIS PASIEN
   - Tambah catatan kunjungan baru
   - Lihat riwayat kunjungan per pasien
   - Edit catatan kunjungan
   - Hapus catatan kunjungan
   
   Data Rekam Medis:
   - ID Rekam Medis (auto-increment)
   - ID Pasien (foreign key)
   - Tanggal Kunjungan
   - Keluhan Utama
   - Diagnosis
   - Catatan Dokter
   - Tindakan yang Dilakukan
   - Tanggal Kontrol Berikutnya
   
   Fitur Tambahan:
   - Timeline riwayat kunjungan
   - Filter berdasarkan tanggal
   - Print rekam medis per pasien


3. RESEP OBAT
   - Tambah resep untuk pasien
   - Lihat riwayat resep per pasien
   - Edit resep
   - Hapus resep
   
   Data Resep:
   - ID Resep (auto-increment)
   - ID Pasien (foreign key)
   - ID Rekam Medis (foreign key)
   - Tanggal Resep
   - Nama Obat
   - Dosis (misal: 10mg, 2 tablet)
   - Frekuensi (misal: 3x sehari, 2x sehari)
   - Durasi (misal: 7 hari, 14 hari)
   - Aturan Pakai (misal: Sesudah makan, Sebelum tidur)
   - Catatan Khusus
   
   Fitur Tambahan:
   - Multi-obat dalam satu resep
   - Template obat umum (untuk input cepat)
   - Print resep
   - Riwayat resep per pasien


4. DASHBOARD
   - Statistik Real-time:
     * Total Pasien Terdaftar
     * Pasien Aktif
     * Pasien Baru Bulan Ini
     * Total Kunjungan Bulan Ini
     * Total Resep Bulan Ini
   
   - Grafik:
     * Grafik jumlah pasien per bulan (6 bulan terakhir)
     * Grafik kunjungan per bulan
     * Diagram pie: Pasien berdasarkan jenis kelamin
   
   - Quick Actions:
     * Tambah Pasien Baru
     * Lihat Daftar Pasien
     * Tambah Rekam Medis
     * Tambah Resep
   
   - Recent Activity:
     * 5 Pasien terakhir terdaftar
     * 5 Kunjungan terakhir


5. EXPORT & PRINT
   - Export Data Pasien ke CSV
   - Export Rekam Medis ke CSV
   - Export Resep ke CSV
   - Print Data Pasien (dengan preview)
   - Print Rekam Medis per Pasien
   - Print Resep
   - Print Laporan Statistik


6. DARK MODE
   - Toggle tema gelap/terang
   - Simpan preferensi user di config file
   - Smooth transition animation
   - Custom color scheme:
     
     Light Mode:
     - Background: #FFFFFF, #F5F7FA
     - Text: #2C3E50
     - Primary: #4A90E2
     - Secondary: #7ED321
     - Accent: #F5A623
     
     Dark Mode:
     - Background: #1E1E1E, #2D2D30
     - Text: #E0E0E0
     - Primary: #5BA3F5
     - Secondary: #8FE63D
     - Accent: #FFB84D


STRUKTUR DATABASE (CSV FILES)
================================================================================

1. pasien.csv
   Format: id,nama,nik,jenis_kelamin,tanggal_lahir,alamat,telepon,riwayat_penyakit,tanggal_registrasi,status

2. rekam_medis.csv
   Format: id,id_pasien,tanggal_kunjungan,keluhan,diagnosis,catatan_dokter,tindakan,tanggal_kontrol

3. resep.csv
   Format: id,id_pasien,id_rekam_medis,tanggal_resep,nama_obat,dosis,frekuensi,durasi,aturan_pakai,catatan

4. config.csv
   Format: key,value
   Contoh: theme,dark
